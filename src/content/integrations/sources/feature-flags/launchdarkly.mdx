<IntegrationSection title="Overview" icon="eye">
LaunchDarkly is a feature management platform built around feature flags and targeting rules.
</IntegrationSection>

<IntegrationSection title="How it works" icon="refresh-cw">
Use LaunchDarkly inside Novu code steps to evaluate feature flag values for a given context and branch your workflow logic based on the result.
</IntegrationSection>

<IntegrationSection title="Configure" icon="settings">
Install the LaunchDarkly SDK, initialize it with your server-side credentials, and evaluate flags inside your Novu workflow code step using the appropriate evaluation context.
</IntegrationSection>
