## Overview

Flagsmith is a feature flag and remote configuration platform for web, mobile, and server-side applications.

## How it works

Use Flagsmith inside Novu code steps to fetch and evaluate feature flags or remote config values and use them to control workflow behavior.

## Configure

Install the Flagsmith SDK, add your environment credentials, and evaluate flags inside your Novu workflow code step.
