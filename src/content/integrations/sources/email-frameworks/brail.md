## Overview

Brail is a React-based toolkit for creating, generating, and delivering HTML emails with built-in tooling for correctness, previews, and type safety.

## How it works

Use Brail inside Novu code steps to generate email HTML from React-based templates.

## Configure

Install Brail, define your email template, and generate the output you want to pass into your Novu workflow logic.
