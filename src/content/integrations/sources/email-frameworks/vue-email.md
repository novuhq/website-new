## Overview

Vue Email is a collection of unstyled components and render utilities for building emails with Vue and TypeScript.

## How it works

Use Vue Email inside Novu code steps to build email templates as Vue components and render them to HTML.

## Configure

Install the Vue Email packages, create your email template with Vue components, and render the template inside your Novu workflow code step.
