## Overview

Maizzle is an HTML email framework powered by Tailwind CSS and email-focused build tooling.

## How it works

Use Maizzle inside Novu code steps to build email templates with HTML and Tailwind-based tooling, then generate production-ready email HTML.

## Configure

Install Maizzle, create your template and config, and run the build process from your Novu workflow code step or supporting build logic to produce the final HTML.
