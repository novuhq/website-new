<IntegrationSection title="Overview" icon="eye">
MJML is a markup language designed to make responsive email easier to build and maintain. Its engine compiles MJML into responsive HTML for email clients.
</IntegrationSection>

<IntegrationSection title="How it works" icon="refresh-cw">
Use MJML inside Novu code steps to write email templates in MJML and compile them into responsive HTML before sending.
</IntegrationSection>

<IntegrationSection title="Configure" icon="settings">
Install mjml, write your template with MJML components, and compile it to HTML within your Novu workflow code step.
</IntegrationSection>
