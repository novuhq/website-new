## Overview

React Email is a collection of unstyled components and utilities for building email templates with React and TypeScript.

## How it works

Use React Email inside Novu code steps to generate email content as HTML from React components.

## Configure

Install the React Email packages, create your email component, and render it to HTML inside your Novu workflow code step.
