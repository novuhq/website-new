<IntegrationSection title="Overview" icon="eye">
  The Vercel AI SDK is a TypeScript toolkit for building AI-powered applications
  and agents. It supports text generation, structured outputs, tool calls, and
  more.
</IntegrationSection>

<IntegrationSection title="How it works" icon="refresh-cw">
  Use the AI SDK inside Novu code steps to generate or transform workflow
  content, including structured outputs when you need predictable JSON.
</IntegrationSection>

<IntegrationSection title="Configure" icon="settings">
  Install the ai package, connect your model provider, and call functions such
  as generateText from your Novu workflow code step.
</IntegrationSection>
