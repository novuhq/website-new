<IntegrationSection title="Overview" icon="eye">
LangChain is a framework for building LLM-powered applications and agents with model and tool integrations.
</IntegrationSection>

<IntegrationSection title="How it works" icon="refresh-cw">
Use LangChain inside Novu code steps to generate, transform, classify, or summarize workflow content with the model provider you choose.
</IntegrationSection>

<IntegrationSection title="Configure" icon="settings">
Install LangChain and the model integration package you need, add your model credentials, and call the model from your Novu workflow code step.
</IntegrationSection>
