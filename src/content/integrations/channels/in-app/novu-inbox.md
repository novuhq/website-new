## Overview

Novu Inbox is a drop-in notification center component for your application. It handles real-time delivery, preference management, read/unread tracking, and action buttons.

## How it works

Add the Inbox component to your frontend. Novu handles real-time delivery, state management, and subscriber preferences without additional backend logic.

## Configure

Install @novu/react or @novu/nextjs and add the Inbox component. No third-party integration required.
